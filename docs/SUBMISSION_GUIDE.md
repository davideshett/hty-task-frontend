# Submission Guide

## Run locally
1. Install deps: `npm install`
2. Add .env file (see variables below)
3. Run dev: `npm run dev`

## Environment variables
VITE_API_BASE= // local backend base (e.g., http://localhost:5000)
VITE_MEMBERBASE_BASE=https://demo-log.memberbase-sandbox.com
VITE_MEMBERBASE_KEY=eyJ... (use sandbox key)

## Recording tips
- Show creating an event in Memberbase
- Show listing on frontend
- Show login & booking
- Show delegate/booking appearing in Memberbase
